## Prompt profile: simple-baseline

Goal: produce a valid `submission.csv` reliably under a short budget, then make at least a small effort to improve the score.

Time management: prioritize reliability. Within the first ~60 seconds, ensure you can generate a valid `submission.csv` end-to-end (create `train_model.py` and run `python train_model.py`).

Execution note: use shell commands to write files and verify `train_model.py` / `submission.csv` exist (avoid IDE/editor tools).

Use the remaining time to try a couple of alternative approaches and keep the one with the best local validation score.

Always leave a valid `submission.csv` behind.

Do not install packages.
