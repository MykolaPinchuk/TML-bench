## Overall (across competitions)

- Ranks are computed per competition using each model/config’s best normalized score (higher-is-better). `mean_rank_pct` is a percentile rank within each competition (0% is best).
- Aggregation is shown separately per `prompt_profile`.

- Absolute signal uses two fixed host baselines per competition: `constant` and `hgb`.
  - `mean_abs_units`: 0 = constant baseline, 1 = hgb baseline (higher is better).
  - `beat_hgb_rate`: fraction of competitions where the model beats the hgb baseline.
