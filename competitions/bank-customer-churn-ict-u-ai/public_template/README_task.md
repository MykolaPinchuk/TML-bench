# Bank customer churn (binary classification)

Your goal is to predict whether a customer will churn (close their account).

## Files
- `train_public.csv` — training data (includes the target column `Exited`)
- `test_public.csv` — holdout features (no labels)
- `sample_submission.csv` — required submission schema

## Target
- `Exited` (binary)

## Output
Write `submission.csv` in the workspace root matching `sample_submission.csv` exactly.
