# applied-regression-on-structured-attributes

Competition scaffold for `applied-regression-on-structured-attributes`.

## Prepare data

This competition appears to require accepting Kaggle rules before the CLI can download data.

```bash
KAGGLE_CONFIG_DIR=secrets python competitions/applied-regression-on-structured-attributes/prepare_competition.py --download
```

If you get a `403 Forbidden`, open the competition page in a browser, accept the rules, then rerun:
`https://www.kaggle.com/competitions/applied-regression-on-structured-attributes`
